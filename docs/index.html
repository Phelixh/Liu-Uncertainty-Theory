Mirrored Modules

The repository intentionally keeps a small curated subset of files that represent the Uncertainty Theory layer under discussion in the PR.

Mathlib.lean

Import snapshot file copied from the source tree.

BaseCore.lean

Core structures and bridge definitions for the uncertainty layer.

BaseDistribution.lean

Distribution-side interfaces, formulas, and special families.

BaseProcess.lean

Process-level structures, renewal updates, and Euler-scheme interfaces.

Base.lean

Aggregate module importing the three base uncertainty modules.

Uncertainty.lean

Top-level public module for the uncertainty namespace.

Sync Workflow

This site and repository are maintained by mirroring the selected files from the upstream PR branch and then refreshing the recorded source head.

Typical Procedure

  1. Update Phelixh/mathlib4 on feat/uncertainty-pr.
  2. Copy the mirrored files into this repository.
  3. Review the diff and refresh the recorded source commit.
  4. Commit and push to main.
